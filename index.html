Project Overview

AtliQ Technologies faced a challenge in understanding employee attendance patterns post-pandemic. This project provides a comprehensive dashboard to track Presence %, Work From Home (WFH) utilization, and Sick Leave (SL) trends.

  • Analyzed 3 months of daily attendance data.
  • Identified key trends in hybrid work adoption.
  • Provided actionable insights for HR resource planning.
